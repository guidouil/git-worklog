# Changelog

Toutes les modifications notables de ce projet sont documentées ici. Le format suit [Keep a Changelog](https://keepachangelog.com/fr/1.1.0/) et le projet suit [Semantic Versioning](https://semver.org/lang/fr/).

## [0.1.0] - 2026-07-22

### Ajouté

- Analyse des périodes civiles et plages ISO dans le fuseau local.
- Sessions configurables avec durée minimale, paddings et découpage à minuit.
- Filtrage d’auteur et fusion de plusieurs dépôts Git.
- Rapports terminal français/anglais, JSON, CSV et dashboard HTML autonome.
- Configuration locale et globale avec priorité des options CLI.
- Statistiques par jour, dépôt, jour de semaine et heure.
- Tests unitaires et tests d’intégration dans des dépôts Git temporaires.
